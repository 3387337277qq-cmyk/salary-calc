# 教师工资计算器 — 部署说明

## 一、准备工作

### 1. 注册 Supabase（免费）
1. 打开 https://supabase.com ，点击 "Start your project"
2. 用 GitHub 账号注册登录
3. 创建一个新项目（选择 Asia Pacific 区域更快的速度）
4. 记下项目 URL 和 anon key

### 2. 初始化数据库
1. 进入 Supabase 项目的 SQL Editor
2. 打开 `supabase/migrations/001_schema.sql` 文件
3. 复制全部内容，粘贴到 SQL Editor 中运行

### 3. 配置环境变量
编辑项目根目录的 `.env` 文件：
```
VITE_SUPABASE_URL=https://你的项目ID.supabase.co
VITE_SUPABASE_ANON_KEY=你的anon密钥
```

### 4. 启动应用
```bash
npm install
npm run dev
```

## 二、部署到 Vercel（免费）
1. 将代码推送到 GitHub 仓库
2. 打开 https://vercel.com ，用 GitHub 登录
3. 导入项目，Vercel 自动检测 Vite 配置
4. 在环境变量设置中添加 `VITE_SUPABASE_URL` 和 `VITE_SUPABASE_ANON_KEY`
5. 部署！

## 三、可选：配置百度 OCR（图片识别）

如果要使用真实的 OCR 识别功能：
1. 注册百度智能云：https://cloud.baidu.com
2. 开通"表格文字识别"服务（每月1000次免费）
3. 获取 API Key 和 Secret Key
4. 部署 Supabase Edge Function：
   ```bash
   npx supabase functions deploy ocr-recognize
   ```
5. 在 Supabase Dashboard 设置环境变量：
   - `BAIDU_OCR_API_KEY`
   - `BAIDU_OCR_SECRET_KEY`

如果不配置 OCR，系统会自动使用模拟数据演示功能。

## 四、PWA 图标

将你的应用图标（192x192 和 512x512 的 PNG）替换 `public/` 目录下的占位文件。

## 五、项目结构

```
salary-calc/
├── src/
│   ├── components/layout/  # 布局组件
│   ├── pages/              # 页面组件
│   ├── lib/                # 工具库（定价计算、Supabase客户端）
│   ├── stores/             # 状态管理
│   └── types/              # TypeScript 类型
├── supabase/
│   ├── migrations/         # 数据库迁移
│   └── functions/          # Edge Functions
└── public/                 # 静态资源
```
